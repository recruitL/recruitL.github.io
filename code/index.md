### py-sc

Python 数值计算课程型代码库，按书籍章节组织理论笔记、Notebook、示例脚本和可复用实现。当前第二章“数据插值”已完成第一轮系统建设，覆盖插值基本形式、Lagrange/Newton 插值、Runge 现象、Chebyshev 节点、分段线性插值和自然三次样条插值，并保留 Hermite、PCHIP、B 样条与二维插值的后续扩展入口。
